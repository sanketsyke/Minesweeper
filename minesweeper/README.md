# Minesweeper CLI

## How to Run

1. Ensure you have Python 3.x installed.
2. Run `python main.py` to start.
3. Follow prompts to reveal (`r row col`), flag (`f row col`), save (`s`), or quit (`q`).
4. Game saves to `data/minesweeper_save.txt`.

## Rules

- Clear all cells that do not contain mines to win.
- Flag all suspected mines.
- Revealing a mine ends the game.

See [Wikipedia: Minesweeper](https://en.wikipedia.org/wiki/Minesweeper_(video_game)) for more details.
